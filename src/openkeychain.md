# OpenKeychain

在本章中，你将学习 OpenKeychain 的下载与安装，以及以下功能的使用：

- 创建 OpenPGP 密钥对
- 导出自己的公钥
- 导入并认证别人的公钥
- 加密文本或文件
- 解密文本或文件
